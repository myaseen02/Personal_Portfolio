# Public Assets Folder

This folder contains static assets that will be served from the root of your website.

## Resume PDF Placement

**IMPORTANT:** Place your resume PDF file here with the exact filename:

```
resume.pdf
```

### Steps:
1. Copy your resume PDF file (`M Yaseen - Resume (NEWEST).pdf`) to this `public` folder
2. Rename it to `resume.pdf` (exactly as shown)
3. The download button on the About page will automatically work

### File Location:
```
public/
  └── resume.pdf  ← Place your resume here
```

The file will be accessible at `/resume.pdf` when the site is running, and the download button will trigger the download with the filename `Mohammad_Yaseen_Resume.pdf`.
